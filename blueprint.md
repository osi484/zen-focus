# Zen Focus - Blueprint

## 프로젝트 개요
Zen Focus는 사용자가 고요한 환경에서 업무나 학습에 몰입할 수 있도록 돕는 웹 기반 생산성 도구입니다. 미니멀한 디자인과 시각적/청각적 요소를 결합하여 최적의 집중 환경을 제공합니다.

## 현재 구현된 기능 및 디자인
- **디자인 컨셉:**
  - 다크 테마 (#0f172a) 기반의 럭셔리 미니멀리즘 UI.
  - 글래스모피즘(Glassmorphism) 및 넓은 여백을 활용한 고급스러운 레이아웃.
  - 탭 제목(Title) 실시간 타이머 반영.
- **주요 기능:**
  - **다국어 지원 (i18n):** 한국어와 영어 자동 감지 및 수동 선택 기능 (localStorage 저장).
  - **집중 타이머:** 커스텀 시간 설정, 시작/일시정지/리셋 기능.
  - **집중 통계:** 타이머 종료 시 누적 집중 시간 기록 및 표시 (localStorage).
  - **앰비언트 사운드:** 최적화된 4종의 로컬 음원 및 볼륨 조절 기능.
  - **고도화된 할 일 목록:** 체크 시 자동 정렬(아래로 이동), 취소선 적용, 완료 항목 전체 삭제.
  - **스마트 알림:** 타이머 종료 시 브라우저 알림 및 하단 팝업 알림 제공.

## 향후 계획
1. **PWA 도입:** 오프라인에서도 사용 가능한 프로그레시브 웹 앱 설정.
2. **다양한 테마:** 사용자 취향에 맞는 여러 가지 미니멀 테마 제공.
3. **통계 시각화:** 주간/월간 집중도를 그래프로 확인하는 기능.
